# Love2D-Stuff
Different apps/games implemented with Lua's LÖVE2D engine.

## Structure
- `love-games/` - Love2D projects (dots, sticks, FractalAnimations, Ping_Pong, pixelating)
- Web gallery at project root (Vite site with listing page)

## Run locally
```bash
love love-games/dots
love love-games/sticks
# etc.
```

## Web
```bash
npm install
npm run dev
```
